## College Student Mental Health Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/15367b623ace439192c3348b7dacf72d/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Teacher End Functions:**
1. Login
2. Homepage
3. Personal Center (Change Password, Personal Information)
4. Health Knowledge Management
5. Notification Management
6. Student Management
7. Psychology Teacher Management
8. Exam Paper Management
9. Exam Management



**Student End Functions:**
1. Login
2. Register
3. Homepage
4. Exam Papers List
5. Health Knowledge
6. Notifications
7. Psychology Teacher
8. Personal Center
9. Backend Management (Homepage, Personal Center, Health Knowledge Management, Notification Management, Psychology Teacher Management, Exam Management)

**Administrator Functions:**
1. Login
2. Homepage
3. Personal Center (Change Password, Personal Information)
4. Basic Data Management (Health Knowledge Type Management, Time Period Management, Notification Type Management)
5. Health Knowledge Management
6. Notification Management
7. Student Management
8. Psychology Teacher Management (Psychology Teacher Management, Psychology Teacher Comment Management, Psychology Teacher Favorite Management, Psychological Consultation Appointment Application Management)
9. Exam Paper Management
10. Exam Management
11. Carousel Management
